## 快速开始

### 1. 安装依赖（已完成）
```bash
cd /Users/edward/workspace/edward/personal-blog
npm install  # ✅ 已完成
```

### 2. 启动开发服务器
```bash
npm run dev
```
访问：http://localhost:4321

### 3. 构建生产版本
```bash
npm run build
```

### 4. 预览生产版本
```bash
npm run preview
```

## 下一步建议

### 📝 添加博客文章
在 `src/pages/blog/` 目录下创建新的 `.astro` 文件：
```bash
touch src/pages/blog/your-post-slug.astro
```

### 🔧 配置内容集合（推荐）
创建 `src/content/config.ts` 来使用 Astro 的内容集合功能，实现类型安全的文章管理。

### 🌐 部署到 Cloudflare Pages
1. 将代码推送到 GitHub
2. 在 Cloudflare Dashboard 创建 Pages 项目
3. 连接 GitHub 仓库
4. 设置构建命令：`npm run build`
5. 设置输出目录：`dist`

### 💬 添加评论系统
集成 [Giscus](https://giscus.app)（基于 GitHub Discussions）

### 🔍 添加搜索功能
集成 [Pagefind](https://pagefind.app) 实现客户端搜索

### 📊 添加网站统计
集成 [Umami](https://umami.is) 或 Google Analytics
